**მოკლე აღწერა**  
Space Quiz Game არის მარტივი Python თამაში, რომელიც მოთამაშეს კოსმოსთან დაკავშირებულ კითხვებს უსვამს. თამაშში გამოიყენება ASCII Art ვიზუალები, პირობითი ოპერატორები, ციკლები, და ფუნქციები.
1   -ascii_art.py-    თამაშის ვიზუალური ნაწილი / welcome screen -> ნიკოლოზ ლონგურაშვილი
2   -quiz_logic.py-   კითხვები, პასუხების შემოწმება, ქულები -> გიორგი გოშაძე
3   -main.py-         მთავარი ლუპი / მომხმარებლის ინტერფეისი -> დათო ვეფხვაძე
